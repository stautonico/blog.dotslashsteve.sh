I'd like to be able to write a book for the [[http://docdock.netlify.com/content-organisation/][docdock theme]]. It says to
nest content like this:
#+BEGIN_EXAMPLE
content
├── level-one
│   ├── level-two
│   │   ├── level-three
│   │   │   ├── level-four
│   │   │   │   ├── _index.md
│   │   │   │   ├── page-4-a.md
│   │   │   │   ├── page-4-b.md
│   │   │   │   └── page-4-c.md
#+END_EXAMPLE
#+END_QUOTE

* Books
** book1
:PROPERTIES:
:EXPORT_HUGO_SECTION: book1
:END:
*** Index
:PROPERTIES:
:EXPORT_FILE_NAME: _index
:END:
Book 1 index.

This type of organization will generate this in the =content/=
directory:
#+BEGIN_EXAMPLE
> tree book1
book1
├── chapter1
│   ├── _index.md
│   ├── section1.md
│   └── section2.md
├── chapter2
│   ├── _index.md
│   ├── section1.md
│   └── section2.md
└── _index.md
#+END_EXAMPLE

- [[/book1/chapter1][Chapter 1]]
